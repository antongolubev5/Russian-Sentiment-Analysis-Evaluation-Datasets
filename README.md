# Russian sentiment analysis evaluation datasets

Russian sentiment analysis evaluation datasets from past competitions are posted in this repository.


## ROMIP-2012

Opinions in direct or indirect speeches extracted from news articles.


## SentiRuEval-2015

[Датасеты](https://github.com/Koziev/NLP_Datasets/tree/master/Samples) используются для тренировки чат-бота.
Они содержат короткие предложения, извлеченные из большого текстового корпуса,
а также некоторые паттерны и словосочетания.
